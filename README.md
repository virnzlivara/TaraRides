# TaraRides
TaraRides

#Installation:
yarn or npm install

#Start the app
yarn start

#Run in device
yarn ios
yarn android

Features:

Home Page :
- Map 
- Next and Previous User
- Show direction
- Navigate to List of Requests
- Navigate to the User Details
- Go to current location


Details 
- Show User Details
- Fetch Address given the longitude and latitude

List
- Show User Details
- Fetch Address given the longitude and latitude
